# Legal documents

Templates for Phase 7 (P7-01..05). **Every document here is a starting
draft, NOT signed-off legal copy.** Before publishing publicly:

1. Send to a UK solicitor (recommend a fintech or marketplace specialist).
2. Customise placeholders: `[Company Name]`, `[Address]`, `[Date]`.
3. Date the version and keep prior versions in this repo for audit.

Once reviewed, copy the final text into the relevant Next.js page:
- ToS → `/terms` (TBD)
- Privacy → `/privacy` (Phase 0 lightweight version is what's live)
- AUP → `/acceptable-use` (TBD)
